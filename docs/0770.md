# 北京协和医学院 - 医药生物技术研究所

## 苗庆芳

学校：北京协和医学院

院系：医药生物技术研究所

* * *

评分：3.5

导师辨识特征：天坛西里1号抗体中心

学术水平：其实还可以

科研经费：科研经费比较充足，对学生的科研需求会极大的满足，比如各种仪器耗材，

学生补助：除了所里补助以外，很少额外发

师生关系：对学生的管理，她会按学生的需求。但相对很宽松，是个很好的导师。不会存在什么抢一作什么的事情，也很少安排杂活

工作时间：不打卡，很随意

学生前途：你强制要求的话应该会允许吧。学术上，发文章比较困难，对以后深造没有太大帮助
