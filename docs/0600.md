# 清华大学 - 核能与新能源技术研究院

## 王海涛

学校：清华大学

院系：核能与新能源技术研究院

* * *

评分：5.0

导师辨识特征：山东人，办公地点在能科楼二楼

学术水平：清华本博，力学专业出身，核科学各方面都水平很高，包括但不限于力学，材料，探测等方面

工作时间：不打卡，也不需要去昌平基地，非常自由

学生前途：原则上不允许实习，但假期都按学校要求放，王老师也会给学生推荐工作
