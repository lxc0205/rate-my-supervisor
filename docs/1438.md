# 西安建筑科技大学 - 土木工程学院

## 门进杰

学校：西安建筑科技大学

院系：土木工程学院

* * *

评分：5.0

自证认识导师：土木楼309

学术水平：科研能力不错，正值事业上升期，发表sci ei多篇，思想能跟得上前沿。

科研经费：可以满足试验和会议要求。

学生补助：按劳发补助。

师生关系：对学生不错，尊重学生自己的想法，鼓励学生自己思考。

工作时间：周一到周五，不打卡。

学生前途：老师没有明确说过实习的问题，学生一般在暑期实习，毕业的学生工作没有问题，老师有时候会推荐工作单位，也有部分学生去更高的学校读博。
